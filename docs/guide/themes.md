# Themes

Themes defines visual identity of your component providing unique layout and feel for the component.

## Available Themes

Dyvix provides a wide range of themes. You can trigger these by passing the string name or the exported constant object for types safety.

- `DYVIX_GLOBAL_THEME.SINGULARITY`: `'Singularity'`
- `DYVIX_GLOBAL_THEME.INDUSTRIAL`: `'Industrial'`
- `DYVIX_GLOBAL_THEME.EMBER`: `'Ember'`
- `DYVIX_GLOBAL_THEME.FROST`: `'Frost'`
- `DYVIX_GLOBAL_THEME.BLADE`: `'Blade'`
- `DYVIX_GLOBAL_THEME.NEON`: `'Neon'`
- `DYVIX_GLOBAL_THEME.AURORA`: `'Aurora'`
- `DYVIX_GLOBAL_THEME.SUNSET`: `'Sunset'`
- `DYVIX_GLOBAL_THEME.OCEAN`: `'Ocean'`
- `DYVIX_GLOBAL_THEME.FOREST`: `'Forest'`
- `DYVIX_GLOBAL_THEME.MIDNIGHT`: `'Midnight'`
